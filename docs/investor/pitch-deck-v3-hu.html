Hogyan működik

  • A munkavállaló anyanyelvén írja meg a jegyét / megjegyzését
  • A rendszer automatikusan lefordítja a viewer preferred_language-ére
  • A magyar admin magyarul lát mindent — nincs copy-paste Google Translate-be
  • Két irány: admin → munkavállaló válasz is anyanyelven jelenik meg

Műszaki részletek

  • Model: claude-haiku-4-5-20251001
  • Prompt caching a system prompton — ~90% input-költség-csökkentés
  • 30 napos DB cache (translation_cache tábla) — ismétlődő szövegek ingyen
  • XML-tagolt input — prompt injection védelem
  • Fallback: Google Translate (ha Claude API nem elérhető)

🦄 Bruno AI — Kontextus-érzékeny Munkatárs-képző

Animált AI-karakter, amely munkafolyamat közben segít — különösen az új külföldi munkavállalóknak

Mit csinál Bruno?

  • Kontextus-érzékeny tippek: látja, melyik oldalon van a user, és célzott tanácsot ad
  • Animált reakciók: motiváló visszajelzés gamification-rendszerrel együtt
  • Anyanyelven beszél: a viewer nyelvi preferenciája szerint
  • Nem tolakodó: csak akkor jelenik meg, ha kell — új user, elakadt folyamat, új funkció

Miért fontos?

Egy filippínó vagy ukrán munkavállaló első napján ritkán kap megfelelő onboarding-et. Bruno beugrik helyette: megmutatja, hogyan adjon be jegyet, hol töltse ki a pulse-felmérést, mit tegyen kárigény esetén.

Technikai háttér

  • components/Bruno/BrunoCharacter.jsx
  • 296 sor CSS — teljes animációs library
  • Admin belépési pont: pages/BrunoTest.jsx
  • Integrálható minden admin- és mobil-képernyőn
  • Claude-alapú válaszgenerálás, ha a user kérdez
Eredmény: rövidebb onboarding, kevesebb HR-kérdés, magasabb adopciós arány — különösen a nyelvi akadályokkal küzdő új munkavállalóknál.
